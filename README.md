## Project Description
This is a **Multi-Threaded Student Grade Processing System** that demonstrates all the concepts covered in the course:
- Encapsulation & OO Relationships (Section III)
- Abstraction, Abstract Class, Polymorphism, Overriding (Section IV)
- Threading, Synchronization, wait()/notify(), Deadlock Prevention (Section V)
- Exception Handling & File I/O (Lecture 4)

## How to Compile and Run
1. Open the project in NetBeans
2. Run `Main.java`
3. Reports will be generated automatically in the `reports/` folder

## Features Implemented
- Full encapsulation and validation
- Abstract class + Polymorphism
- Multi-threading (Thread + Runnable)
- Thread synchronization with wait()/notify()
- Deadlock prevention (ordered locks)
- Custom exceptions
- File I/O using Scanner & PrintWriter
